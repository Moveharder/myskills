## 上报类型列表

### 渠道说明
渠道定义和说明：
- `活动首页电视广告机`: 电视广告机
- `活动首页社群广告图`: 社群广告图
- `活动首页小程序banner`: 小程序banner
- `活动首页share`: 分享链接
- `活动首页DEV`: 开发环境（默认值）

## 上报流程

### 问答流程
1. 用户进入Fajo首页 → 根据入口渠道上报：
   - 电视广告机 → `活动首页电视广告机`
   - 社群广告图 → `活动首页社群广告图`
   - 小程序banner → `活动首页小程序banner`
   - 分享链接 → `活动首页share`
   - 其他/开发环境 → `活动首页DEV`
2. 用户依次回答4道问题 → `问答1`、`问答2`、`问答3`、`问答4`
3. 用户提交答案 → `问答完毕`
4. 跳转到结果页 → `问答结果`（或`问答分享被打开`如果通过分享进入）
5. 用户可进行领券或分享操作 → `问答结果页领券`、`问答结果页分享`

### 抽奖流程
1. 用户查看抽奖结果 → `抽奖结果`
2. 用户点击领券 → `抽奖结果页领券`

## 文件位置
- ftyCoFajo.js: 活动首页页面
- question.js: 问答页面
- qaResult.js: 问答结果页面
- lotteryResult.js: 抽奖结果页面
