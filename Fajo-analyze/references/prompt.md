# USER

当前目录是分析「Fajo-2026步履新生」活动数据分析工作空间。
- event-describe.md文件：定义了活动数据上报类型，包括：首页访问的渠道来源、问答数据、问答结果分享数据、抽奖结果数据等等；
- event-report.json文件：是活动数据上报文件，里面包含用户行为数据，包括：用户ID、上报时间、上报类型；提示：如果没有找到该文件，请用户提供文件地址或直接提供数据。

你需要按照event-describe.md文件中的描述，使用Python或JavaScript等语言，对event-report.json文件进行解析，并生成可视化报表。

如果已经存在分析脚本（如：analyze.py），请勿重复编写。

报表细节：
- 需考虑图标涉及到的中文符号，避免乱码；
- 主要关注指标：0、总事件数和每个事件数（图表按照数量降序排序）、1.渠道占比（渠道类型：活动首页小程序banner, 活动首页电视广告机, 活动首页社群广告图, 活动首页share, 活动首页DEV）、2.参与答题人数、3.完成答题人数、4.问答结果页分享次数、6.问答分享被打开次数、7.领券次数（问答结果页领券和抽奖结果页领券）；
- 需要同时生成可视化图表和数据报表，数据报表格式为Markdown；
- 可视化图表上需要标注出每个事件对应的具体值，涉及到排序，请务必按照数量降序排序；
- 所有数据分析内容存放在一个文件夹中，文件夹名称为活动名称+数据日期范围，如：Fajo2026步履新生-2026-03-02至2026-03-03；
